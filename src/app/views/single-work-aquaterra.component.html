Aquaterra

Website re-design & build

A mobile first responsive website design and build for a charitable sports and leisure business.

The previous site suffered from crucial user experience issues including navigation difficulty, broken user journeys and replicated yet orphaned content. The re-design of the site utilised a simple and responsive layout with a core colour palette to communicate functional journeys and page information. The site shrinks right down to a single column layout so that mobile users can easily navigate the site without compromising on legibility of content. A website built to adapt to all digital environments.

Leveraging technology to help make community participation easier.
