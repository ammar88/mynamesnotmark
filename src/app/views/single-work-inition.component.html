Inition

Website re-design & build

Website re-design for a pioneering interactive production company and emerging technology provider.

Inition commisioned Firedog to create an engaging and content-focused user experience, featuring impactful video content and imagery, as well as a simple and confident use of typography. Accessing and using the site on mobile devices was also a necessity. All pages were built for optimal results in any environment. On mobile devices, imagery and typography are optimised for comfortable viewing experiences.

The site is easy to use on mobile - button spacing and search features allow for easy navigation.
